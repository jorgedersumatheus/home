      # BLACKVOX Playlists
## Descrição
O aplicativo BLACKVOX permite criar, salvar, carregar e gerenciar playlists de áudio.
### Funcionalidades
- Adicionar Áudio
- Salvar Playlist
- Carregar Playlists
- Importar Playlists de Arquivo
#### Como Usar
1. Adicione áudios através do botão "Baixar Áudio".
2. Crie e salve suas playlists.
3. Carregue playlists existentes e importe playlists de arquivo.
##### Tecnologias Utilizadas
- HTML
- CSS
- JavaScript
